# MenuMatematic
MENÚ MATEMÀTIC
**************
Som en Roger i en Sergi de 1r de DAW, i aquest és el nostre programa el MENÚ MATEMÀTIC. Per començar a utilitzar-lo simplement executa l'arxiu "MenuMatematic.exe".

INSTRUCCIONS
************
1. Selecciona una de les opcions de l'1 al 8 del menú.
2. Introdueix els paràmetres que requireix el programa per executar     l'operació.
3. Un cop tens el resultat prem qualsevol tecla per tornar al menú.
4. Pots tornar al pas número 1 i sel·leccionar una opcicó, o bé prémer 'q' i sortir del menú matemàtic
